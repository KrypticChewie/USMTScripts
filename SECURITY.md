# Security Policy

## Supported Versions

Security updates are currently supported for version 1.4.x and higher

| Version | Supported          |
| ------- | ------------------ |
| 1.4.x   | :white_check_mark: |
| 1.2.x   | :x:                |
| 1.1.x   | :x: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

If a vulnerability is found it can be added a bug to the repository.
